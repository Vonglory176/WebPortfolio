React.js Documentation

This is a little documentation page built to give the reader some information on the React Javascript framework. In building it, most all attention was on media queries and bootstrap use to make the page's content completely responsive and scalable. It has 3 main viewports in which the navbar will shift around the screen, the mobile view in particular condensing it to a small hamburger menu. Each link within said navbar is connected to a certain section of the page's content, this also incorperating some basic JavaScript/jQuery to keep the navbar out of the way of headers. Note that the content images included were left on the larger side to better demonstrate this.

Van Gogh Art Gallery

This project is a small art gallery to showcase the paintings of Vincent van Gogh, and was my first swing at trying out CSS's Grid system. I struggled a bit with the row/column calculations in particular and some of the content doesn't wrap as perfectly as I'd like as such, but I thought it was incredibly interesting and had a lot of fun making it. It has several different responsive viewport configurations and content will shift and reorganize as the page narrow/widens.
